# YopalProducts

Aplicativo web para la ciudad de Yopal Casanare, permite publicar y buscar productos de primera necesidad de peque√±as empresas

  - Angular
  - Firebase
  - HTML
  - CSS3

# Funcionalidades!

  - Cargar imagenes de forma estructurada
  - Desplegar una vista de producto seleccionado
  - Interfaz intuitiva
  - Publicacion de productos
  - Interfaz de aprobacion de publicaciones
  - Multiples opciones de productos
